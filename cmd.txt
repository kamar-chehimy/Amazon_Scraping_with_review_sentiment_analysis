1. To create environment and scrapy project type the following
python -m venv scraping_env
scraping_env\Scripts\activate.bat
scrapy startproject Scraping_Amazon

2. To install required libraries
pip install -r requirements.txt

3. change your working directory and run the scraping process that will be saved in csv file
cd Scraping_Amazon
scrapy crawl amazon_scraping -o results.csv
